# 五張撲克牌遊戲

## 專案簡介

使用 **C# Windows Forms (.NET 6)** 開發的五張撲克牌遊戲，玩家可設定押注金額，
根據牌型賠率計算中獎金額。牌面使用 GDI+ 自行繪製，無需外部圖片資源。

## 功能介紹

| 功能 | 說明 |
|------|------|
| 💰 押注 | 輸入押注金額，從總資金扣除 |
| 🃏 發牌 | 從 52 張牌中隨機發 5 張 |
| 🔄 換牌 | 勾選想換的牌後點換牌（可多張） |
| 🏆 判斷牌型 | 自動判斷牌型並結算中獎金額 |

## 賠率表

| 牌型 | 賠率 |
|------|------|
| 皇家同花順 | 250x |
| 同花順 | 50x |
| 四條 | 25x |
| 葫蘆 | 9x |
| 同花 | 6x |
| 順子 | 4x |
| 三條 | 3x |
| 兩對 | 2x |
| 一對 | 1x |

## 執行環境

- **IDE**：Visual Studio 2022
- **Framework**：.NET 6 Windows Forms
- **OS**：Windows 10 / 11

## 執行說明

1. 開啟 `PokerGame.sln`（或直接開啟 `PokerGame.csproj`）
2. 按 **F5** 或點選「▶ 啟動」執行
3. 在「押注金額」輸入想下注的金額，點「押注」
4. 點「發牌」開始遊戲
5. 勾選想換的牌後點「換牌」（可選擇不換牌）
6. 點「判斷牌型」查看結果與中獎金額
